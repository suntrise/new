Windows Classic（Windows 经典主题）是 Windows 部分版本的内置主题。于 Windows 95 起作为 Windows 的默认主题，Windows XP 时被 Luna 主题取代，并于 Windows 8 正式移除。

## 综述
Windows Classic 的主要特征为灰色背景，以及突出的，棱角分明的按钮等控件。
